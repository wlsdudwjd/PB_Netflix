대화 요약 (요청 → 작업/답변)
1) 로그인/회원가입 페이지: 이메일 형식 검증, TMDB API 키 비밀번호 사용, Remember me/자동 로그인, 로컬 스토리지 저장, 성공/실패 메시지, 라우팅 보호(/signin 리다이렉트) 구현.
2) UI/효과: 로그인-회원가입 전환 효과, 토스트/알럿, 넷플릭스풍 스타일, 카드 호버 확대, 찜 토글, 애니메이션 적용.
3) 홈(/): TMDB 섹션(트렌딩/인기/상영중/평점/개봉예정), 히어로 배너, 카드 클릭 시 상세 이동, 찜 토글.
4) 대세 콘텐츠(/popular): 테이블/인피니트 토글, 페이지네이션/무한스크롤, Top 버튼, 탭 네비(홈/대세), 테이블 2×5→반응형 그리드, 모바일 레이아웃/스크롤 수정, 인피니트 전용 카드 디자인/찜 버튼 리디자인, 로고 클릭 홈 이동.
5) 검색(/search): 필터(장르/연도/국가/관람가), 정렬(인기/평점/개봉일), 자동 적용, 인트로 화면 “어떤 영화를 검색하고 싶나요?”, 결과는 검색 후 표시. 제목 포함 검색, 개봉일 조건 제거, 등급 필터 수정(전체관람가 등), 국가/등급 필터 버그 수정.
6) 상세(/movie/:id): TMDB 상세/크레딧/추천 영화 표시, 출연진 아바타에서 이니셜 제거, 추천 카드 클릭 시 같은 뷰 내에서 재로딩되도록 watch 추가, 찜 토글 유지.
7) 위시리스트(/wishlist): 로컬 스토리지 기반, MovieCard 재사용, 레이아웃/간격/정렬(왼쪽부터, 일정 카드 폭, 모바일 대응) 조정.
8) 인증 상태: 전역 라우터 가드로 미인증 시 /signin 리다이렉트. 로그아웃 시 세션 삭제 + 자동로그인만 해제(아이디 저장 유지).
9) 브랜딩: 탭 제목 JINFLIX로 변경, 로고 클릭 시 홈으로 이동, 파비콘 교체 방법 안내.
